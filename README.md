# 🔐 Cybersecurity Homelab (Red + Blue Team Lab)

## 📌 Overview
This project demonstrates a self-hosted cybersecurity lab built using Proxmox to simulate real-world enterprise environments. 

The lab is designed to practice:
- Penetration testing (Red Team)
- Detection and monitoring (Blue Team)
- Active Directory attacks
- Network segmentation and defense

---

## 🧱 Technologies Used
- Proxmox VE (virtualization)
- pfSense (firewall/router)
- Kali Linux (attacker machine)
- Windows Server (Active Directory)
- Windows 10/11 (client)
- Wazuh (SIEM/logging)
- Metasploitable (vulnerable VM)

---

## 🌐 Lab Architecture
- Segmented internal network
- Domain environment (corp.local)
- Centralized logging via Wazuh
- Controlled traffic via pfSense

---

## 🎯 Objectives
- Simulate real-world cyber attacks
- Practice detection and response
- Build hands-on cybersecurity experience

---

## 🧪 Skills Demonstrated
- Network scanning and enumeration
- Exploitation and initial access
- Credential harvesting and password attacks
- Privilege escalation
- Lateral movement in Active Directory
- Log analysis and threat detection
